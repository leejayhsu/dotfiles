# dotfiles

### config.lua
This is for [LunarVim](https://www.lunarvim.org)
To set up the symlink, do (this assumes you cloned this to your home directory), and you have already installed LunarVim:
```shell
ln -sf ~/dotfiles/config.lua ~/.config/lvim
```
